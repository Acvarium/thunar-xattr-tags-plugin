thunar-xattr-tags-plugin
====================

 The Thunar Mattr Tags Plugin (thunar-xattr-tags-plugin) adds special features to add tags to files in the Thunar File Manager with xattr.

From source: 

    % cd thunar-xattr-tags-plugin
    % ./autogen.sh
    % make
    % make install
